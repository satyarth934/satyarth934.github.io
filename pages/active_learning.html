Active Learning

By: Satyarth Praveen, Tarushree Gandhi, and Manish Soni

Active Learning is a form of semi-supervised learning in the field of machine learning.

As per the general trend for supervised machine learning models, the requirement of large amounts of data is critical, not raw data, but well-annotated data. Though most of the times we have an abundant pile of data, getting it accurately annotated is a manual, cumbersome task that takes up a lot of effort and time. This issue is precisely where Active Learning comes into play. Using some of the smart techniques of this concept, we can select only the most complex and informative data samples for annotation. This step exponentially slashes down the manual work by sampling only a small fraction from the original set of data samples.

Following are some of the many pitfalls for using the entire data set:

  • With a lot of similar data samples in the training set, one might cause the model to overfit.
  • An imbalance between the hard data samples compared the simple ones, might hamper the learning model from converging towards the global minima.
  • The investment in the manual annotation time is much more than the improvements we get in return.

The data collection units from the company send over vast piles of data. So I designed and implemented a pipeline to process the new incoming data, and categorize it as "useful" and "not useful." Training is performed using the annotated samples of only the "useful" data.

This module has effectively brought down the manual efforts to almost one-third or less. It has also improved the performance of learning models because of a better balance between the hard and easy data samples and reduced training time because of less number of training images compared to the original data set.

The major sub-domains where we have been using this technique are Object Detections and Semantic Segmentation. A modified version of this is also used to identify the diversity of selected sample set and help maintain the balance of varied environments in the learning set.
